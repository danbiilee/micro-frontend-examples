# Next - React

## Getting Started

```bash
yarn && cd host && yarn && cd ../app1 && yarn && cd ../app2 && yarn && cd ../ && yarn start
```

- Host
  - `./host`: localhost:3000
- Remote
  - `./app1`: localhost:3001
  - `./app2`: localhost:3002
