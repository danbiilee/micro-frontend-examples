# React - React

## Getting Started

```bash
yarn bootstrap && yarn start
```

- Host
  - `./host`: localhost:3000
- Remote
  - `./remotes/app1`: localhost:3001
  - `./remotes/app2`: localhost:3002

## Features

- live reloading: `@module-federation/fmr` 사용
- 모노레포 적용: Yarn Workspace + Lerna
- Remote app1에서 shared 패키지 사용
